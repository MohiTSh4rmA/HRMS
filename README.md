# HRMS Lite

## Project Overview
HRMS Lite is a lightweight full-stack Human Resource Management System built as part of a company technical assessment.  
The application focuses on core HR operations such as employee management and attendance tracking with a clean, professional interface.

---

## Features

### Employee Management
- Add a new employee (Employee ID, Name, Email, Department)
- View list of all employees
- Delete an employee
- Validation for required fields and duplicate employees

### Attendance Management
- Mark daily attendance (Present / Absent)
- View attendance records

### UI & UX
- Clean and professional admin interface
- Clear error handling and user feedback
- Simple and intuitive navigation

---

## Tech Stack

### Frontend
- React (JavaScript)
- Fetch API for backend communication

### Backend
- FastAPI (Python)
- RESTful API design
- Pydantic for data validation

### Database
- In-memory storage (for simplicity as per assignment scope)

### Deployment
- Frontend: Render (Static Site)
- Backend: Render (Web Service)
